## Problem 2

**冲突1**

```
|ID + ID + ID -> .....
E + E| + ID
```

既可以移入`+`号，也可以规约`E + E` 为 `E`.

**冲突2**

```
|ID + ID[ID] -> ......
E + E｜[ID] 
```

到这里既可以对`E+E`进行规约， 也可以把`[` 移入.

#### **解决办法**

结合性：`+`, `[]` 均规定为左结合。

优先性：`+ < []` 

1. 对于冲突1，由于加法是左结合，所以只能先进行规约。
2. 对于冲突2，由于优先性`+ < []`，所以只能先进行移入。

**解决办法：**

优先性：`{} > IF E THEN S ELSE S > IF E THEN S `
